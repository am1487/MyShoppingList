2.1 Είσοδος στην Ιστοσελίδα

      Με την είσοδο στην ιστοσελίδα έχετε τη δυνατότητα είτε να δημιουργήσετε μια νέα λίστα για ψώνια πατώντας το κουμπί «Νέα Λίστα» στο κέντρο της οθόνης, είτε να κάνετε σύνδεση με τον λογαριασμό σας (Λογαριασμός πανεπιστήμιου Θεσσαλίας) και να ενεργοποιήσετε τις δυνατότητες συνδεμένων χρηστών οι οποίες αναφέρονται παρακάτω.

2.2 Δημιουργία νέας λίστας

      Μετά το πάτημα «Νέα Λίστα» εμφανίζεται η οθόνη επιλογής περιοχής. Επιλέξτε την περιοχή στην οποία θα ψωνίσετε στο χάρτη και πιέστε το κουμπί «επόμενο».

      Στην επόμενη οθόνη που εμφανίζεται καλείστε να προσθέσετε τα προϊόντα στη λίστα. Αυτό γίνεται πατώντας πάνω στο πεδίο «Προσθήκη» και γράφοντας το όνομα του προϊόντος το οποίο επιθυμείτε να προσθέσετε, θα εμφανιστεί μια λίστα από προϊόντα για να διαλέξετε το κατάλληλο, έπειτα επιλέγετε την ποσότητα από το προϊόν που επιθυμείτε. Μπορείτε σε αυτό το στάδιο να αφαιρέστε ένα προϊόν πατώντας στο «Χ». Μόλις έχετε τελειώσει πατάτε «Υποβολή»

2.3 Προεπισκόπηση λίστας

      Στην οθόνη της λίστας η λίστα μπορεί να χωριστεί σε μέχρι 3 διαφορετικά καταστήματα διαλέγοντας πάντα τις φθηνότερες λύσεις. Μπορείτε να αλλάξετε την επιλογή σας χρησιμοποιώντας τις καρτέλες πάνω από τη λίστα. Στην οθόνη φαίνεται το τελικό πόσο σύμφωνα με τα στοιχεία για το συγκεκριμένο κατάστημα, καθώς και τα καταστήματα στα οποία αντιστοιχεί η κάθε λίστα

      Στην καρτέλα «Προτάσεις» μπορείτε να δείτε μια λίστα από προτάσεις οι οποίες μπορούν να κάνουν την λίστα σας οικονομικότερη (π.χ. κάποιο ανταγωνιστικό προϊόν το οποίο βρίσκεται σε προσφορά).

      Εάν θέλετε να επεξεργαστείτε τη λίστα μπορείτε να πατήσετε στη καρτέλα «Επεξεργασία».

      Ένας συνδεμένος χρήστης μπορεί να αποθηκεύσει τη λίστα του για να τη βρει αργότερα πατώντας το κουμπί «Αποθήκευση».

      Μπορούμε να εκτυπώσουμε τη λίστα σε εκτυπώσιμη μορφή πατώντας στο κουμπί «Εκτύπωση».








      Πηγαίνοντας με το ποντίκι πάνω από κάποιο προϊόν μπορείτε να δείτε κάποιες πληροφορίες για το προϊόν αυτό, όπως την τιμή του σε κάθε διαφορετικό κατάστημα, καθώς και κάποια στατιστικά για τη μεταβολή της τιμής του.

2.4 Συνδεμένος χρήστης

      Ένας συνδεμένος χρήστης έχει τη δυνατότητα να ενημερώσει την τιμή κάποιου προϊόντος αν το βρήκε σε διαφορετική τιμή από την τιμή της υπηρεσίας μάς, πατώντας στο κουμπί «Ενημέρωση τιμής»








      Μετά το πάτημα «Ενημέρωση τιμής» εμφανίζεται η φόρμα ενημέρωσης τιμής για τα προϊόντα. Γράψτε το όνομα του προϊόντος στο πεδίο «Προσθήκη» επιλέξτε το από τα προϊόντα που θα εμφανιστούν και έπειτα γράψτε την νέα τιμή στο πεδίο «Νέα Τιμή», Αυτό μπορεί να γίνει για πολλά προϊόντα στην ίδια φόρμα.

      Πατώντας στο όνομα σας (π.χ. Βασίλης) εμφανίζεται η σελίδα του λογαριασμού χρήστη σας όπου μπορείτε να δείτε τα στοιχεία του προφίλ σας πατώντας στο κουμπί «προφίλ», να δείτε τις αποθηκευμένες λίστες σας πατώντας στο κουμπί «λίστες» και να δείτε τα στατιστικά από τα ψώνια σας (Μέσο λογαριασμό, έξοδα μήνα, έξοδα βδομάδας, Συνήθη αγορασμένα προϊόντα κλπ.) πατώντας στο κουμπί «Στατιστικά».

3 Android

      Εφόσον εγκαταστήσετε την εφαρμογή Android Και συνδεθείτε στο λογαριασμό σας,

      Μπορείτε εύκολα να ενημερώσετε κάποια τιμή είτε μέσω του Barcode του προϊόντος και εισάγοντας τη νέα τιμής πατώντας «Ενημέρωση Τιμής με Barcode», είτε σαρώνοντας την απόδειξη πατώντας «Σάρωση απόδειξης».

      Επίσης ο μπορείτε να δείτε τις αποθηκεύμενες λίστες σας πατώντας στο κουμπί «Οι λίστες μου»

      Στην οθόνη με τη λίστα μπορείτε να τσεκάρετε τα προϊόντα τα οποία έχετε βάλει στο καλάθι πατώντας πάνω στο προϊόν.

      Η να πατήσετε το κουμπί «Ενημέρωση Τιμής» ώστε να ενημερώσετε την τιμή κάποιου προϊόντος
